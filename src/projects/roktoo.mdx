## Features
- Anyone can register and become a blood donator instantly
- Donator's can control which area they are available to donate
- Donator's  can set their availability to donate blood
- Anyone can search for blood by group/district/area

## Tech Stack
- React
- Next
- Node
- Tailwindcss
- NextAuth 
- Custom Authentication
